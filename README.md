# 🔢 Поиск простых чисел с использованием многопроцессорности

## 📖 Описание проекта

Данный проект представляет собой консольную программу на **Python**, которая выполняет **поиск простых чисел в заданном диапазоне** с использованием **многопроцессорности (multiprocessing)**.

Основная цель программы — ускорить вычисления за счёт **распараллеливания задачи между несколькими процессами**, используя все доступные ядра процессора.

Каждый процесс вычисляет простые числа в своей части диапазона, а отдельный процесс отвечает за **сохранение результатов в файлы**.

---

# ⚙️ Как работает программа

Работа программы разделена на несколько этапов:

1. Пользователь вводит число **N**, до которого необходимо найти простые числа.
2. Программа определяет **количество доступных ядер процессора**.
3. Пользователь выбирает, **сколько процессов использовать**.
4. Диапазон чисел делится на части между процессами.
5. Каждый процесс:
   - проверяет числа на простоту
   - отправляет найденные простые числа в очередь (`Queue`)
6. Отдельный процесс (**save_worker**) получает результаты из очереди и сохраняет их в текстовые файлы.
7. После завершения вычислений программа выводит итоговую статистику.

---

# 🧠 Что такое простое число

**Простое число** — это число больше 1, которое делится только на:

- 1
- само себя

Примеры простых чисел:

```
2, 3, 5, 7, 11, 13, 17, 19
```

---

# 🧩 Основные функции программы

## `is_prime(n)`

Функция проверяет, является ли число простым.

Алгоритм:
- если число меньше 2 → не является простым
- проверяется делимость до √n

---

## `find_primes(start, end, result_queue)`

Функция, выполняемая в отдельном процессе.

Задачи:
- перебирает числа в диапазоне
- проверяет каждое число на простоту
- добавляет найденные простые числа в список
- отправляет список в очередь `result_queue`

---

## `save_worker(result_queue, stop_event)`

Отдельный процесс для **сохранения результатов**.

Функции:
- получает списки простых чисел из очереди
- сохраняет их в текстовые файлы
- выводит информацию о сохранении

Файлы создаются автоматически:

```
primes_result_0.txt
primes_result_1.txt
primes_result_2.txt
```

Каждый файл содержит найденные простые числа.

---

# ⚡ Использование multiprocessing

Программа использует модуль:

```
multiprocessing
```

Это позволяет:

- запускать несколько процессов одновременно
- использовать несколько ядер CPU
- ускорять вычисления

Основные элементы:

- `Process` — создание процесса
- `Queue` — обмен данными между процессами
- `Event` — сигнал завершения работы
